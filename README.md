# Macro Cup 2026 - نظام التسجيل الرياضي الاحترافي

هذا التطبيق مخصص لإدارة وتسجيل المشاركين في الفعاليات الرياضية، وهو جاهز تماماً للنشر ليعمل على مدار الساعة (24/7).

## كيفية تشغيل التطبيق أونلاين (Deployment)

لجعل التطبيق متاحاً للجميع عبر رابط دائم، اتبع الخطوات البسيطة التالية:

1. **رفع الكود إلى GitHub:**
   - قم بإنشاء مستودع (Repository) جديد على حسابك في GitHub.
   - ارفع جميع ملفات هذا المشروع إليه.

2. **إعداد Firebase App Hosting:**
   - توجه إلى [Firebase Console](https://console.firebase.google.com/).
   - اختر مشروعك الحالي.
   - من القائمة الجانبية، اختر **App Hosting**.
   - اضغط على "Get Started" وقم بربط حساب GitHub الخاص بك.
   - اختر المستودع والفرع (branch) الذي رفعت عليه الكود.

3. **النشر التلقائي:**
   - سيقوم Firebase ببناء التطبيق ونشره تلقائياً. أي تعديل ترفعه لاحقاً على GitHub سيتم تحديثه في الموقع فوراً.

## التكلفة (Pricing)
- التطبيق يعمل ضمن **الخطة المجانية (Spark Plan)**.
- يوفر Firebase حدوداً مجانية كافية جداً (50,000 عملية قراءة و20,000 عملية كتابة يومياً).
- لن تحتاج لدفع أي مبالغ طالما أن الاستخدام في حدود الفعالية الرياضية المعتادة.

## مميزات النظام
- **تسجيل ذكي**: يمنع تكرار "كود مايسترو" أو "الرقم القومي".
- **حدود التسجيل**: يغلق التسجيل تلقائياً في رياضة ضربات الجزاء عند وصول العدد إلى 32 مشاركة.
- **لوحة تحكم (Admin)**: محمية برمز وصول وتسمح بتحميل البيانات بصيغة Excel (CSV).
- **تصميم متجاوب**: يعمل بكفاءة عالية على الهواتف الذكية وأجهزة الكمبيوتر.

## ملاحظة للمسؤولين
يمكن الوصول للوحة التحكم من خلال الصفحة الرئيسية باستخدام رمز الوصول المخصص للمسؤولين (Admin Code).
