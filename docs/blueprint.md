# **App Name**: Sports Registration Hub

## Core Features:

- User Registration: Allows users to register with their name, department, Maestro code, national ID, contact, gender, and sport of choice.
- Sport Selection: Enables users to select a sport from a predefined list (running, football, penalty).
- Option Selection: For certain sports, allows users to choose from a list of available options. Automatically removes options as they are selected by other users, maintaining uniqueness.
- Validation: Verifies user inputs, checks for duplicate Maestro codes and sport options, and ensures that sport-specific choices are valid.
- Success Notification: Informs users whether their registration was successful or if any errors occurred during the process.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to give the sense of importance and focus
- Background color: Very light Indigo (#F0F8FF), almost white, for a calm, clear interface.
- Accent color: Violet (#8A2BE2), a brighter accent from the same part of the spectrum.
- Body and headline font: 'PT Sans', a humanist sans-serif font that's both modern and approachable, and that is readable at various sizes
- Use clear, simple icons to represent different sports and form fields.
- A clean, organized layout with clearly labeled fields and a straightforward registration process.
- Subtle transition effects to guide users through the registration process smoothly.
